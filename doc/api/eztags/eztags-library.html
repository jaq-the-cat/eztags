Classes

Tag
Represents an ID3v2 tag, the class you will be using the most with this library. type represents a tag type, and data represents the tag's data. A tag of type TagType.artwork has data which represents a URL to an image A tag of type TagType.duration has data which represents the track's duration in milliseconds The other tags are represented by strings which will get encoded into ISO 8859-1/Latin-1.
TagList
List of tags. Does not allow for multiple tags with the same type
